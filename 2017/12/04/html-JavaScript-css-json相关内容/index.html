html

  1. 如何在js中调用jsp脚本的变量?
    在JS中调用JAVA变量可以，方法是：var JS变量名 = <%=JAVA变量名 %>

    <%中间写java代码，跟在JSP中一样%>

    在JAVA中 ，无法调用JS变量

    在一个JSP页面中，先编译java代码，再编译JS代码，所以当java代码编译完毕，渲染页面后就得到了HTML和JS，然后正确执行
